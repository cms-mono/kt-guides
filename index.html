전체 개요 및 진행 흐름

커뮤니즈에서 알림톡/친구톡을 발송하려면 카카오톡 발신프로필 등록이 선행되어야 합니다. 등록 과정은 카카오 측 채널 준비(Part A)와 커뮤니즈 측 발신프로필 등록(Part B) 두 단계로 구성됩니다.

사전 준비물

  • 카카오 계정 (이메일 또는 카카오톡 계정)
  • 사업자등록증 또는 고유번호증
  • 카카오톡 전자증명서 제출이 가능한 휴대폰 (심사용)
  • (해당 시) 업종별 인허가 서류
  • (대행사인 경우) 업무 대행 계약서, 대행사 사업자등록증
  • 채널 관리자로 초대된 사람의 카카오톡 연락처 (토큰 인증 수신용)
  • 커뮤니즈 계정 및 카카오 비즈메시지 서비스 신청 완료 상태
참고 카카오 측 영역은 KT / Communis / Mono의 직접 지원 범위가 아닙니다. 심사 소요 시간, 반려 사유 등 카카오 정책 관련 문의는 카카오 비즈니스 고객센터(https://center-pf.kakao.com)로 문의 바랍니다.

Part A. 카카오 비즈니스 채널 만들기

카카오 비즈니스 사이트(https://business.kakao.com)에서 채널을 생성하고 비즈니스 심사를 받는 과정입니다.

1카카오 비즈니스 로그인

  1. https://business.kakao.com/ 접속
  2. 카카오 계정으로 로그인 (계정이 없다면 회원가입 클릭 후 안내 따름)
카카오 비즈니스 로그인 페이지 (이메일 또는 카카오계정 로그인)

2새 채널 만들기 — 유형 선택

  1. 좌측 하단의 + 새 채널 만들기 클릭
  2. 채널 유형 선택
    • 유형 1. 기본 채널형 — 카카오톡 프로필 채널(채팅·메시지·소식·쿠폰)
    • 유형 2. 오프라인 매장형 — 카카오맵 매장 등록까지 포함
  3. 다음 클릭
알림톡/친구톡 목적이라면 유형 1. 기본 채널형으로 충분합니다.

3사업자 정보 입력

  1. 네, 지금 입력할게요 선택
  2. 사업자등록번호(또는 고유번호) 입력 후 다음
비즈니스 심사를 받으려면 사업자 정보 입력이 필수입니다.

4채널 기본 정보 입력

채널 프로필 사진, 이름, 카테고리, 검색용 ID를 입력합니다.

항목입력 가이드
프로필권장 640×640, JPG/JPEG/PNG, 최대 10MB
채널 이름한글/영문/숫자 20자 이내 (친구 100명 이하일 때 1회 변경 가능)
카테고리사업 업종에 맞는 대분류/소분류 선택
검색용 ID한글/영문/숫자 15자 이내 — 한 번 정하면 변경 불가
검색용 ID는 나중에 커뮤니즈에 @검색용ID 형태로 등록합니다.
주의 검색용 ID는 변경 불가합니다. 오타·임시값 금지, 발송 운영에 사용할 최종 아이디로 입력하세요.

5비즈니스 심사 신청

채널을 만든 직후에는 아직 일반 채널 상태입니다. 발신프로필을 발급받으려면 비즈니스 채널로 전환해야 하고, 그 전환 수단이 비즈니스 심사입니다.

  1. 좌측 메뉴 채널 > 비즈니스 심사 이동
  2. 우측 상단 심사 신청하기 클릭
  3. 요구 서류 업로드 후 제출
필요 서류: 사업자등록증 · 전자증명서 제출용 휴대폰 · (해당 시) 인허가 서류 · (대행사) 계약서·사업자등록증
TIP 서류에 기재된 내용(상호, 업종 등)이 채널 정보(채널명·소개글·프로필 사진)와 일치하는지 확인하세요. 불일치 시 심사가 반려될 수 있습니다.
안내 심사 소요 시간·반려 사유는 카카오 정책에 따라 수시 변경됩니다. 최신 정보는 https://center-pf.kakao.com 공지사항을 참고 바랍니다.

6심사 승인 후 — 검색용 아이디 확인

  1. 심사 승인 완료 후, 채널 > 채널 정보 이동
  2. 검색용 아이디 값을 복사 (예: 모노테스트122)
노란 박스 영역의 값을 복사 → 커뮤니즈 등록 시 앞에 @를 붙여 사용합니다.

Part B. 커뮤니즈에 발신프로필 등록

카카오 비즈니스 채널이 심사 승인되었다면, 이제 커뮤니즈에 @검색용아이디를 등록하고 발신키를 발급받습니다.

1커뮤니즈 로그인 및 메뉴 이동

  1. https://communis.kt.co.kr 접속 후 로그인
  2. 상단 메뉴의 대시보드 버튼 클릭 (로그인 직후 메인 페이지에서는 좌측 메뉴가 보이지 않음)
  3. 대시보드 진입 후 좌측 메뉴에서 발신정보 > 카카오 발신프로필 클릭
참고 로그인만 한 상태(메인 페이지)에서는 좌측 네비게이션이 나타나지 않습니다. 반드시 상단 대시보드 메뉴로 먼저 진입해야 좌측 메뉴가 표시됩니다.
발신정보 메뉴 하위의 카카오 발신프로필
상단 안내사항 확인 후, 우측 + 발신 프로필 등록 버튼 클릭

2발신 프로필 등록 — 입력

세 항목을 정확히 입력한 뒤 토큰 요청 클릭
항목입력 값
카카오 채널 ID @검색용아이디 형식으로 입력 (예: @모노테스트122)
※ 반드시 @ 를 앞에 붙여야 합니다.
관리자 연락처 해당 카카오 채널의 관리자로 초대된 사람의 휴대폰 번호.
※ 이 번호로 카카오톡 인증 토큰이 전송됩니다.
사업자 카테고리 대분류 → 중분류 → 소분류를 순서대로 선택
주의 관리자 연락처는 카카오 채널 관리자로 등록된 사람의 번호여야 합니다. 채널 관리자가 아닌 번호를 넣으면 인증 토큰 발송이 실패합니다.

3토큰 요청 및 카카오 인증

  1. 토큰 요청 버튼 클릭
  2. 관리자 연락처로 카카오 알림톡으로 6자리 인증 토큰 수신
  3. 목록의 해당 프로필 우측 인증 버튼 클릭 → 6자리 입력 → 확인
초기 상태: 발신프로필 차단, 발신키 미인증 — 우측 인증 클릭
카카오톡으로 받은 6자리 숫자 입력 → 확인

4발신 키 발급 완료

카카오 인증이 완료되면 발신키(발신 프로필 키)가 발급됩니다. 단, 아래 두 상태가 모두 정상이어야 실제 발송에 사용할 수 있습니다.

발신키가 부여되고 상태가 정상 / 정상으로 표시됨
상태 컬럼의미정상 값
발신프로필 상태커뮤니즈 내부 승인 상태정상
카카오톡 채널 상태카카오 측 채널 승인 상태정상
중요 카카오 토큰 인증은 즉시 승인되지만, 커뮤니즈 내부에서 추가 승인이 완료되어야 발신프로필 키를 실제 발송에 사용할 수 있습니다. 등록 직후 발신이 안 되면 상태를 확인 바랍니다.
참고 발신 프로필 등록 직후에는 일별 최대 발송량이 1,000건으로 제한됩니다. 이후 운영 실적에 따라 상향됩니다.
